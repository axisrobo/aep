TRADEMARKS

Trademarks

Protocol Name

"Harmovela Protocol" is a working name for this open coordination protocol. The name is provisional and may evolve before the 1.0 release.

Axisrobo

"Axisrobo" is a trademark of the Axisrobo project. The Axisrobo name and logo may not be used in ways that suggest endorsement of third-party implementations, derivatives, or services without explicit permission.

Usage Guidelines for Community Implementations

Protocol Conformance

Protocol conformance does not require trademark usage. Implementations may conform to the Harmovela Protocol without using the Harmovela or Axisrobo names in their branding, provided they accurately describe their conformance status.

Fair Use of the Protocol Name

Community implementations and tools may use the term "Harmovela" in descriptive contexts to indicate compatibility:

  • "compatible with Harmovela Protocol"
  • "supports Harmovela subscriptions"
  • "implements Harmovela task lifecycle"

These descriptive uses must be truthful and must not imply endorsement by or affiliation with Axisrobo.

Naming Community Implementations

Community-maintained implementations, SDKs, libraries, and tools should:

  • Choose their own distinct project and product names
  • Avoid names that could be confused with official Axisrobo implementations
  • Clearly identify themselves as community projects, not produced by Axisrobo

Acceptable examples:

  • "Acme Event Bus — a community SDK compatible with Harmovela Protocol"
  • "Foo Agent Runtime (Harmovela-compatible)"
  • "Bar Memory Bridge — community implementation of Harmovela memory events"

Unacceptable:

  • "Harmovela SDK by Acme" (implies official status)
  • "Axisrobo Agent Runtime" (uses Axisrobo trademark without permission)

Third-Party Identification

Third-party implementations, forks, and derivatives must clearly identify themselves as not produced by or affiliated with Axisrobo. A notice such as the following should be included in the project's README or equivalent:

> This project is an independent community implementation and is not produced by, endorsed by, or affiliated with the Axisrobo project or the Harmovela Protocol maintainers.

Open Source Use

The Harmovela Protocol specifications, schemas, and conformance fixtures are openly licensed. Implementations may use the protocol definitions freely under the terms of the Apache 2.0 license. Trademark usage, however, is governed by this document, not by the open source license.

Changes

Trademark policy may be updated as the project moves toward independent governance. Material changes will be announced through the project's communication channels.
